# 🧊 Smart Fridge Meal Suggestion System

## Overview

The Smart Fridge Meal Suggestion System is a web-based application designed to help users manage food inventory efficiently, reduce food waste, and simplify meal planning. The system allows users to track available food items, monitor expiry dates, receive low-stock alerts, generate grocery lists, and discover meal suggestions based on ingredients currently available in their refrigerator.

The project was developed as part of the Information System Design (ISD) course to demonstrate the application of software engineering principles, system analysis, and modern web development technologies in solving real-world household management problems.

## Problem Statement

Many households purchase food items that eventually expire before being consumed. Poor inventory tracking often leads to food waste, unnecessary grocery purchases, and difficulty deciding what meals can be prepared using available ingredients.

The Smart Fridge Meal Suggestion System addresses these challenges by providing an intelligent inventory management platform that helps users keep track of food items and make informed meal decisions.

## System Workflow

1. User logs into the system.
2. User adds food items to the virtual fridge inventory.
3. The system stores item details and expiry information.
4. Inventory levels are continuously monitored.
5. The system identifies items nearing expiration.
6. Meal recommendations are generated using available ingredients.
7. Missing ingredients are automatically added to a grocery list.
8. Users update inventory as food items are consumed.
